FoodieFleet – Online Food Delivery SQL Project

FoodieFleet is a step-by-step SQL project simulating an online food delivery platform, built progressively over six days. It includes schema design, data insertion, and business-focused queries using MySQL.

Files by Day

- `Day1.sql` – Schema creation: Tables, primary/foreign keys, and relationships
- `Day2.sql` – Sample data insertion for all tables
- `Day3.sql` – Basic SELECT queries and filters
- `Day4.sql` – JOINs, grouping, and aggregation queries
- `Day5.sql` – Subqueries, nested logic, and conditional queries
- `Day6.sql` – Advanced business queries and insights (e.g., top items, delivery stats)

Database Includes

- Customers, Restaurants, and Food Items
- Orders, Order Status, Delivery Partners
- Payments, Ratings, and Order Tracking

Sample Insights

- Most ordered food items by category
- Restaurants with highest customer ratings
- Delivery partner performance
- Revenue by city or food type

Tools Used

- MySQL 8.0+
- MySQL Workbench


This project is public for academic learning, project showcasing, and portfolio building.
